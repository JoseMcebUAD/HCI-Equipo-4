# AI PROMPT

In the process of designing usability testing tasks, I decided to use ChatGPT to generate the collecting data tasks for the module, the usabilty attribute i decided to focus on was efficiency in order to obtain task completion time, number of errors, and steps required to complete each task. Additionally, I specified that each task should include a clear context, goal, and success criteria in order to obtain structured and useful results.

The answer generated by the AI was appropriate for the initial data, as it proposed common tasks for this type of system, such as creating, modifying, or canceling appointment. These tasks were well-structured and aligned with the efficiency attribute, making it easier to visualize how the metrics could be collected, the only problem i could identify was that it tried to give an unrealistic amount of success percentage with the sample it had.

When comparing this process to the manual design of tasks carried out, the difference in time was significant. While the manual work took several hours due to discussion, validation, and refinement, creating the prompt and obtaining results from the AI took only a few minutes. 

I think that the best strategy atleast by the way i work is using the AI Tool in order to create the metrics and an initial success criteria that can help us measure when we do the usability testing


"I am designing a usability test for a scheduling system . The target users are the administrative staff that is going to do the appointments. I want to evaluate the usability attribute of efficiency (how quickly users can complete tasks). Based on this, generate a set of realistic usability testing tasks."
