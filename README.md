# HCI — Team 4
## Agenda Module · UADY Psychology Clinic

This repository contains the user-centered design documentation for the **Agenda module** of ClinicaWeb, a digital system for the UADY Psychology Clinic. The module enables administrative staff (secretaries and coordinators) to manage patient appointments — scheduling, editing, canceling, and processing requests.

---

## Deliverable Contents

| Folder | Description |
|--------|-------------|
| `UserResearch/` | User research process: objectives, methodology, user profiles, key findings, and design recommendations |
| `UserModeling/` | User modeling: primary persona and usage scenarios for the Agenda module |
| `ProductRequirements/` | Functional and usability requirements derived from the research |
| `ProjectDefinition/` | Project scope, social relevance, innovation rationale, and feasibility analysis |
