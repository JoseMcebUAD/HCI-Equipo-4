# HCI — Team 4
## Agenda Module · ClinicaWeb · UADY Psychology Clinic

This repository contains the user-centered design documentation and the functional HTML prototype for the **Agenda module** of ClinicaWeb, a digital management system for the UADY Psychology Clinic. The module enables administrative staff (secretaries and coordinators) to manage patient appointments — scheduling, editing, canceling, and processing requests.

### 🚀 Second Delivery: Prototype & Usability

For this second delivery, our focus shifted from initial documentation to a working, testable prototype. We refined our usability requirements, established measurable metrics, and ensured every requirement is traced directly to the interface design.
